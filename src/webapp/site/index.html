Timegrid

Timegrid is a DHTML-based AJAXy widget for visualizing time-based events. It is capable of rendering dynamic, calendar-style views of event data from multiple sources of different formats (XML, JSON). Below is a live example that you can play with.

The data file for this Timegrid: month.xml

Just like Google Calendar, Timegrid can be used with zero software installation, server-side or client-side. Timegrid, however, does not depend on any server for its data backend; instead, it can import events from simple XML or JSON files, or optionally be configured to display Google Calendar feeds.

Live Examples

  • Conference Schedule: A 3-day schedule of events at a conference, taken from ISWC 2005.

Related Projects

  • Exhibit: a lightweight structured data publishing framework that is entirely Javascript-based—you can make nice time lines and maps without ever touching a database or even setting up a web server.
  • Timeline: a DHTML-based AJAXy widget for visualizing time-based events. It is like Google Maps for time-based information.
  • Timeplot: a DHTML-based AJAXy widget for visualizing time data as time series graphs.
